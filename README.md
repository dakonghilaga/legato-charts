# Legato Charts
### API Services for building products and services about music artists.

## Development Prerequisites

* Code editor of choice
* Docker https://www.docker.com/
* PNPm https://pnpm.io/

## Install dependencies

```bash
pnpm i
```

## Setup Husky Git Hooks

```bash
pnpm run prepare
```

## Start the project

```bash
pnpm run start
```
